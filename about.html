About Aura

Aura is a simple companion app that combines an empathetic AI chat with a lightweight reminder system. It uses a small Flask backend to securely relay your messages to Google’s Gemini model and Firebase/Firestore to store your reminders.

How it works

  • Chat requests go from your browser → Flask → Gemini → back to your browser.
  • Reminders are stored directly in Firestore under your anonymous user ID.
  • All UI is built with Tailwind CSS for clarity and speed.

Tech

  • Frontend: HTML + Tailwind + Firebase Web SDK
  • Backend: Python Flask + CORS + requests
  • AI: Gemini 1.5 Flash
  • Database: Firestore
